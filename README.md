# Flask MongoDB POC

**API Endpoints**

<ol>
<li>Create user => localhost:5000/user</li>
<ul>
Request Body => 
`{
    "name": "Anmol Shinde",
    "email": "anmolshinde106@gmail.com",
    "password": "12345"
}`
</ul>
<li>Get user => localhost:5000/user/<id></li>
<li>List all users => localhost:5000/users</li>
<li>Delete a user => localhost:5000/user/<id></li>
</ol>
